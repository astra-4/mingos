creating my own OS ^-^

a personal website i built using html, css, and javascript. i made it for my beautiful wife :3. maybe i'll turn it into an app later for her.

features:
- any additions you add will save even if you reload or exit the page - it's stored locally though so others won't be able to see it
- cute little loading bar
- welcome page - simple introduction that pops up when you click "MingOS" on top left corner, first taskbar app, or upon loading
- diary (star log) - a space for personal journal entries - you can add your own!
- doodles - a gallery for drawings and artwork - you can add your own!
- mixtape - a music playlist section, for now i have placeholder audio but i'll add my own later, maybe even the music that i'm working on right now hehe
- sketch - draw directly in browser
- guestbook - leave messages and notes - i don't have a database so unfortunately all the data will stay local
- Sunny - a virtual pet and interactive companion to care for while exploring the site - maybe i'll add an adjustment for need levels later

running:
- use https://astra-4.github.io/mingos/
